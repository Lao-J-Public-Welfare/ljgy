Windows 发展时间线

1985
Windows 1.0 – 首个图形界面版本，基于MS-DOS，支持鼠标操作。
1987
Windows 2.0 – 支持重叠窗口，改进内存管理。
1990
Windows 3.0 – 巨大成功，引入程序管理器、文件管理器。
1995
Windows 95 – 开始菜单、任务栏、即插即用，革命性更新。
1998
Windows 98 – 集成IE，改进USB支持。
2000
Windows Me – 消费版末代，基于DOS。
2001
Windows XP – NT内核，稳定经典，统治桌面多年。
2007
Windows Vista – 引入Aero，但硬件要求高。
2009
Windows 7 – 优化Vista，成为经典。
2012
Windows 8 – Modern UI，取消开始菜单。
2013
Windows 8.1 – 回归开始按钮。
2015
Windows 10 – 统一平台，开始菜单回归。
2021
Windows 11 – 全新UI，支持安卓应用。
